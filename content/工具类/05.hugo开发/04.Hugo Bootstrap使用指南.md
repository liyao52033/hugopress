# Hugo Bootstrap使用指南

本指南介绍了在Hugo项目中使用Bootstrap框架的方法，包括组件初始化、常用组件使用和自定义配置。

## Bootstrap集成方式

### 1. 模块导入配置

在`themes/lotusdocs/config.toml`中配置Bootstrap模块导入：

```toml
[module]
[[module.imports]]
path = "github.com/twbs/bootstrap"
```

### 2. 资源管道处理

通过Hugo资源管道处理Bootstrap资源：

```html
<!-- 在layouts/partials/basic.html中 -->
{{ $bootstrapJS := resources.Get "js/bootstrap.js" }}
{{ $bootstrapJS = $bootstrapJS | js.Build (dict "targetPath" "js/bootstrap.bundle.min.js" "minify" true) }}
<script src="{{ $bootstrapJS.RelPermalink }}" defer></script>
```

### 3. SCSS变量自定义

在`assets/scss/_variables.scss`中自定义Bootstrap变量：

```scss
// Bootstrap变量覆盖
$primary: #2563eb;
$secondary: #64748b;
$success: #16a34a;
$info: #0ea5e9;
$warning: #f59e0b;
$danger: #dc2626;

// 导航栏自定义
$navbar-padding-y: 0.5rem;
$navbar-dark-color: rgba($white, .85);
$navbar-dark-hover-color: $white;
```

## Bootstrap组件初始化

### 1. 核心组件导入

在`assets/docs/js/bootstrap.js`中导入和初始化Bootstrap组件：

```javascript
// 导入Bootstrap组件
import Tab from "/js/bootstrap/src/tab";
import Collapse from "/js/bootstrap/src/collapse";
import Dropdown from "/js/bootstrap/src/dropdown";
import ScrollSpy from "js/bootstrap/src/scrollspy";
import Tooltip from "js/bootstrap/src/tooltip";

// 注册到全局作用域
window.Collapse = Collapse;
window.Dropdown = Dropdown;
window.Tooltip = Tooltip;

// 初始化所有下拉菜单
document.addEventListener('DOMContentLoaded', function() {
    const dropdownElementList = document.querySelectorAll('.dropdown-toggle');
    const dropdownList = [...dropdownElementList].map(dropdownToggleEl => new Dropdown(dropdownToggleEl));
});
```

### 2. 工具提示初始化

```javascript
// 初始化工具提示
document.addEventListener('DOMContentLoaded', function() {
    // 初始化所有工具提示
    const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
    const tooltipList = [...tooltipTriggerList].map(tooltipTriggerEl => new Tooltip(tooltipTriggerEl));
});
```

## 自定义Bootstrap行为

### 1. 响应式导航栏增强

在`assets/js/responsive-navbar.js`中添加自定义导航栏行为：

```javascript
document.addEventListener('DOMContentLoaded', function() {
    const navbarToggler = document.querySelector('.navbar-toggler');
    const navbarCollapse = document.querySelector('.navbar-collapse');
    
    if (navbarToggler && navbarCollapse) {
        // 检查Bootstrap是否可用
        if ((typeof bootstrap === 'undefined' || !bootstrap.Collapse) && 
            (typeof window.Collapse === 'undefined')) {
            
            // 自定义折叠行为
            navbarToggler.addEventListener('click', function(e) {
                e.preventDefault();
                const isExpanded = navbarToggler.getAttribute('aria-expanded') === 'true';
                
                if (isExpanded) {
                    navbarCollapse.classList.remove('show');
                    navbarToggler.setAttribute('aria-expanded', 'false');
                } else {
                    navbarCollapse.classList.add('show');
                    navbarToggler.setAttribute('aria-expanded', 'true');
                }
            });
        }
    }
    
    // 点击外部关闭移动菜单
    document.addEventListener('click', function(e) {
        if (navbarCollapse && navbarCollapse.classList.contains('show')) {
            const isClickInside = navbarCollapse.contains(e.target) || 
                                navbarToggler.contains(e.target);
            
            if (!isClickInside) {
                navbarCollapse.classList.remove('show');
                navbarToggler.setAttribute('aria-expanded', 'false');
            }
        }
    });
});
```

### 2. 工具提示增强

```javascript
// 增强工具提示功能
function initTooltips() {
    const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
    const tooltipList = [...tooltipTriggerList].map(tooltipTriggerEl => {
        return new Tooltip(tooltipTriggerEl, {
            delay: { show: 500, hide: 100 },
            placement: 'top'
        });
    });
}

// 页面加载和导航后初始化工具提示
document.addEventListener('DOMContentLoaded', initTooltips);
```

## 最佳实践

1. **组件初始化**：确保在DOM加载完成后初始化Bootstrap组件
2. **资源优化**：使用Hugo资源管道压缩和合并Bootstrap资源
3. **响应式设计**：利用Bootstrap的响应式工具类创建适配多种设备的布局
4. **自定义样式**：通过SCSS变量覆盖Bootstrap默认样式，保持设计一致性
5. **渐进增强**：为不支持JavaScript的环境提供基本功能

## 故障排除

### 1. 组件不工作

- 确保Bootstrap JS文件正确加载
- 检查组件初始化代码是否在DOM加载后执行
- 验证HTML结构是否符合Bootstrap要求

### 2. 样式不生效

- 确认Bootstrap CSS文件正确引入
- 检查SCSS编译是否成功
- 验证自定义变量是否正确覆盖默认值

### 3. 响应式问题

- 确认视口meta标签正确设置
- 检查容器类使用是否正确
- 验证断点设置是否符合需求

通过以上指南，您可以在Hugo项目中有效使用Bootstrap框架，创建响应式、交互性强的网站界面。
